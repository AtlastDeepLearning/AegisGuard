# AegisGuard

A unified threat defense platform combining Phishing Detection (PhishGuard) and Network Intrusion Detection (Sentinet).

## Features

- Phishing URL scanning
- Real-time Network Traffic Anomaly Detection
- AI-powered Security Reporting (Planned)

## Setup

1. `python -m venv .venv`
2. `pip install -r requirements.txt`
3. `uvicorn backend.app.main:app --reload`
